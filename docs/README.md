## Project Overview

The **Sanzo Color Advisor** is an AI-powered color consultation system based on Sanzo Wada's legendary "Dictionary of Color Combinations" from 1918. It provides:

- **348 Traditional Japanese Colors** - Authentic Sanzo Wada palette
- **AI-Powered Recommendations** - Smart color matching for any space
- **Room-Specific Analysis** - Tailored advice for different room types
- **Accessibility Built-in** - WCAG AAA compliance checking
- **High Performance** - 300k+ color operations per second
